Drop your own image files into this folder using these exact filenames
and the site will use them automatically instead of the stock placeholders:

  bg-parallax.jpg     -> global parallax background (site-wide)
  gallery1.jpg        -> Gallery: VR Combat Zone photo
  gallery2.jpg        -> Gallery: Racing Simulators photo
  gallery3.jpg        -> Gallery: PC Gaming Arena photo
  gallery4.jpg        -> Gallery: Console Lounge photo
  gallery5.jpg        -> Gallery: Fuel Bar photo
  gallery6.jpg        -> Gallery: Front Counter photo
  testimonial1.jpg    -> Reviewer avatar #1
  testimonial2.jpg    -> Reviewer avatar #2
  testimonial3.jpg    -> Reviewer avatar #3

Recommended sizes: 1920x1080+ for bg-parallax.jpg, 800x600+ for gallery
photos, square (300x300+) for testimonial avatars.
